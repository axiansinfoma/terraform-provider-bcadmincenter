# Data Source (bcadmincenter_timezones)

Retrieves the list of valid time zone identifiers that can be used for environment update window settings.

This data source retrieves the list of valid Windows time zone identifiers that can be used when configuring environment update windows. Each timezone includes information about its display name, UTC offset, and daylight saving time support.

## Example Usage

### Basic Usage

```terraform
# Copyright (c) 2025 Michael Villani
# SPDX-License-Identifier: MPL-2.0

# Query available time zones
data "bcadmincenter_timezones" "available" {}

# Output all timezones
output "all_timezones" {
  value = data.bcadmincenter_timezones.available.timezones
}

# Find a specific timezone
locals {
  pacific_timezone = [
    for tz in data.bcadmincenter_timezones.available.timezones :
    tz if tz.id == "Pacific Standard Time"
  ][0]
}

output "pacific_timezone_info" {
  value = {
    id           = local.pacific_timezone.id
    display_name = local.pacific_timezone.display_name
    offset       = local.pacific_timezone.offset_from_utc
    has_dst      = local.pacific_timezone.supports_daylight_savings
  }
}

# Use in environment settings
resource "bcadmincenter_environment_settings" "example" {
  application_family = "BusinessCentral"
  environment_name   = "production"

  update_window_start_time = "22:00"
  update_window_end_time   = "06:00"
  update_window_timezone   = local.pacific_timezone.id
}
```

### Find Specific Timezone

```terraform
data "bcadmincenter_timezones" "available" {}

locals {
  # Find Central European Standard Time
  cet_timezone = [
    for tz in data.bcadmincenter_timezones.available.timezones :
    tz if tz.id == "Central European Standard Time"
  ][0]
}

output "cet_info" {
  value = local.cet_timezone
}
```

### Use with Environment Settings

```terraform
data "bcadmincenter_timezones" "available" {}

# Use timezone in environment settings
resource "bcadmincenter_environment_settings" "prod" {
  application_family = "BusinessCentral"
  environment_name   = "production"

  update_window_start_time = "21:00"
  update_window_end_time   = "03:00"
  # Use a timezone from the data source
  update_window_timezone   = data.bcadmincenter_timezones.available.timezones[0].id
}
```

## Common Time Zones

Some commonly used time zone identifiers include:

- `Pacific Standard Time` - (UTC-08:00) Pacific Time (US & Canada)
- `Mountain Standard Time` - (UTC-07:00) Mountain Time (US & Canada)
- `Central Standard Time` - (UTC-06:00) Central Time (US & Canada)
- `Eastern Standard Time` - (UTC-05:00) Eastern Time (US & Canada)
- `GMT Standard Time` - (UTC+00:00) Dublin, Edinburgh, Lisbon, London
- `Central European Standard Time` - (UTC+01:00) Belgrade, Bratislava, Budapest, Ljubljana, Prague
- `W. Europe Standard Time` - (UTC+01:00) Amsterdam, Berlin, Bern, Rome, Stockholm, Vienna
