# Resource (bcadmincenter_environment_support_contact)

Manages the support contact information for a Business Central environment. This is the contact information displayed to users in the Help and Support page.

The support contact information is displayed to users in the **Help and Support** page within Business Central. This allows partners and administrators to provide customized support contact details for their Business Central environments.

## Important Notes

~> **Warning:** Deleting this resource will only remove it from Terraform state. The Business Central Admin Center API does not provide a DELETE endpoint for support contacts, so the contact information will remain configured on the environment. To clear the support contact, you must manually update it in the Business Central Admin Center or set it to different values.

-> **Note:** The support contact is environment-specific. Each environment can have its own support contact configuration.

## Best Practices

### Contact Information Management

1. **Keep Contact Information Current**: Ensure the email and URL are actively monitored and up-to-date
2. **Use Dedicated Support Email**: Use a dedicated support email address rather than personal addresses
3. **Provide Helpful URLs**: Link to comprehensive support portals or knowledge bases
4. **Environment-Specific Contacts**: Consider different contacts for production vs. sandbox environments

### Multi-Tenant Scenarios

When managing multiple tenant environments:

```terraform
# Use variables for consistent contact information across environments
variable "support_contacts" {
  description = "Support contact configuration per environment"
  type = map(object({
    name  = string
    email = string
    url   = string
  }))
  default = {
    Production = {
      name  = "Production Support"
      email = "prod-support@example.com"
      url   = "https://support.example.com/prod"
    }
    Sandbox = {
      name  = "Dev Support"
      email = "dev-support@example.com"
      url   = "https://support.example.com/dev"
    }
  }
}

resource "bcadmincenter_environment_support_contact" "environments" {
  for_each = var.support_contacts
  
  application_family = "BusinessCentral"
  environment_name   = each.key
  
  name  = each.value.name
  email = each.value.email
  url   = each.value.url
}
```

## Common Issues

### Contact Not Visible to Users

If users don't see the updated contact information:
- Ensure users refresh their Business Central session
- Verify the environment name matches exactly (case-sensitive)
- Check that users have the necessary permissions to view the Help and Support page

### Cannot Remove Contact Information

The API does not support deleting support contacts. To work around this:
- Update the contact to generic values before destroying the Terraform resource
- Manually clear the contact in the Business Central Admin Center if needed

### Email Validation

The API validates email format. Ensure:
- Email addresses follow standard format (user@domain.com)
- Email domain exists and is properly configured
- No special characters that might be rejected by validation
